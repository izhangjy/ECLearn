│ 3.1.0 单元学习入口                                                                                                   │
│                                                                                                                      │
│ 首页 → 英语 → 选择年级/学期 → 选择单元 → 单元学习页                                                                  │
│ 单元学习页包含：                                                                                                     │
│ - 本单元单词（数量、已掌握数）                                                                                       │
│ - 本单元语法点                                                                                                       │
│ - 本单元阅读训练                                                                                                     │
│ - 单元综合测试（可包含之前单元知识点）                                                                               │
│ - 单元掌握度进度条                                                                                                   │
│                                                                                                                      │
│ 3.1.1 单词精通（按单元组织）                                                                                         │
│                                                                                                                      │
│ - 单元单词列表：每单元30-50个核心词汇                                                                                │
│ - 学习模式：                                                                                                         │
│   - 认识模式：单词卡片（词、音标、释义、例句、派生词）                                                               │
│   - 拼写模式：听音写词、看义写词                                                                                     │
│   - 语境模式：在句子中选择正确单词                                                                                   │
│ - 刻意练习机制：                                                                                                     │
│   - 掌握标准：同一单词连续3次正确才算掌握                                                                            │
│   - 智能复习：基于遗忘曲线，推送历史单元单词复习                                                                     │
│   - 错词强化：错误单词自动加入强化队列                                                                               │
│   - 跨单元复习：可选择复习任意历史单元的单词                                                                         │
│                                                                                                                      │
│ 3.1.2 语法攻克（单元+专题双维度）                                                                                    │
│                                                                                                                      │
│ - 单元语法：每单元对应的语法重点                                                                                     │
│ - 专题语法：15大高考语法专题（跨单元汇总）                                                                           │
│   - 时态语态、主谓一致、情态动词                                                                                     │
│   - 定语从句、名词性从句、状语从句                                                                                   │
│   - 非谓语动词、虚拟语气、倒装句等                                                                                   │
│ - 刻意练习机制：                                                                                                     │
│   - 单元模式：学完本单元语法后测试                                                                                   │
│   - 专题模式：针对某一语法点集中训练（汇集所有单元相关题目）                                                         │
│   - 综合模式：混合多单元、多语法点                                                                                   │
│                                                                                                                      │
│ 3.1.3 阅读理解（单元主题+题型双维度）                                                                                │
│                                                                                                                      │
│ - 单元阅读：与单元主题相关的阅读材料                                                                                 │
│ - 题型专练：细节理解、推理判断、主旨大意、词义猜测、七选五                                                           │
│ - 刻意练习机制：                                                                                                     │
│   - 限时训练：模拟考试时间压力                                                                                       │
│   - 错题回顾：分析错误选项的干扰点                                                                                   │
│                                                                                                                      │
│ 3.1.4 单元综合测试                                                                                                   │
│                                                                                                                      │
│ - 测试范围：本单元单词+语法+阅读                                                                                     │
│ - 可选范围：勾选"包含之前单元知识点"进行综合复习                                                                     │
│ - 测试结果：生成单元掌握报告，标记薄弱点                                                                             │

│ 2.3.1 试卷导入                                                                                                       │
│                                                                                                                      │
│ - 拍照识别：调用云端OCR识别试卷内容                                                                                  │
│ - 相册导入：支持多张图片拼接                                                                                         │
│ - 手动录入：OCR识别不准时可手动修正                                                                                  │
│ - 科目识别：自动判断英语/语文试卷                                                                                    │
│                                                                                                                      │
│ 2.3.2 智能错题分析（AI驱动）                                                                                         │
│                                                                                                                      │
│ - 知识点定位：AI自动识别错题对应的知识点                                                                             │
│ - 错因诊断：                                                                                                         │
│   - 知识盲点：该知识点从未掌握                                                                                       │
│   - 理解偏差：概念理解有误                                                                                           │
│   - 审题失误：题目理解错误                                                                                           │
│   - 粗心大意：会做但做错                                                                                             │
│ - 详细解析：AI生成错题的详细解答过程                                                                                 │
│ - 关联分析：找出相关联的其他薄弱知识点                                                                               │
│                                                                                                                      │
│ 2.3.3 变式训练生成（刻意练习核心）                                                                                   │
│                                                                                                                      │
│ - 同类变式：AI生成与错题同知识点、同题型的变式题                                                                     │
│ - 难度递进：                                                                                                         │
│   - Level 1：基础巩固（比原题略简单）                                                                                │
│   - Level 2：同等难度（与原题相当）                                                                                  │
│   - Level 3：拔高挑战（比原题更难）                                                                                  │
│ - 举一反三：拓展到相关知识点的练习                                                                                   │
│ - 掌握判定：连续答对3道变式题才算掌握该知识点                                                                        │
│                                                                                                                      │
│ 2.3.4 错题本管理                                                                                                     │
│                                                                                                                      │
│ - 智能分类：按科目→知识点→错因自动归类                                                                               │
│ - 复习提醒：基于遗忘曲线推送复习                                                                                     │
│ - 掌握追踪：显示每个知识点的掌握状态                                                                                 │
│ - 统计分析：错题分布、高频错误知识点                                                                                 │
